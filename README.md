# MRI
## Сегментация опухолей на МРТ с использованием YOLOv8

Программа для автоматического обнаружения и сегментации опухолей на МРТ-изображениях.  
Визуализация выполняется в виде зелёных контуров вокруг обнаруженных объектов.  
Результаты (метаданные, координаты рамок и контуров) сохраняются в базу данных SQLite для дальнейшего анализа.

### Возможности

- Выбор папки с изображениями через графический диалог.
- Обработка изображений форматов JPG, PNG, BMP, TIFF.
- Сегментация опухолей с помощью YOLOv8 (модель `yolov8n-seg.pt`).
- Отрисовка контуров сегментации на изображении.
- Сохранение обработанных изображений в подпапку `results`.
- Сохранение структурированных данных в SQLite БД (имя файла, путь, количество объектов, время обработки, координаты рамок, уверенности, контуры).

### Требования

- Python 3.8+
- Установленные библиотеки: `opencv-python`, `ultralytics`, `numpy`, `pillow`, (встроенные: `sqlite3`, `tkinter`).
